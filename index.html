CurvPy Documentation

Welcome to the CurvPy documentation! CurvPy is a powerful data analysis tool for performing regression analysis and optimization on datasets. This library is designed to simplify the process of analyzing and optimizing data, making it accessible to both beginners and experienced data scientists.

Installation

pip install curvpy

Usage

Regression Analysis

CurvPy provides a simple interface for performing regression analysis on your data. Here's an example of how to use CurvPy for regression analysis:


import numpy as np
from curvpy import datasleuth

# Example usage
x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 6, 8, 10])

# Perform regression analysis
results = datasleuth(x, y)
print(results)

Optimization

In addition to regression analysis, CurvPy also offers an optimization module for fitting functions to your data. Here's an example of how to use CurvPy for optimization:


import numpy as np
from curvpy import optifit

# Define the function to fit
def func(x, a, b, c):
    return a * np.sin(b * x) + c

# Generate some example data
x_data = np.linspace(0, 10, 100)
y_data = func(x_data, 2.5, 1.3, 0.5) + np.random.normal(scale=0.5, size=len(x_data))

# Initial guess for parameters
guess_params = [1.0, 1.0, 1.0]

# Perform curve fitting and display results
fit_results = optifit(func, x_data, y_data, guess_params)
print(fit_results)
