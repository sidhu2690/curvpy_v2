CurvPy Manual

optimize

Fits a model function to data using curve fitting. Allows specifying bounds and maximum evaluations.

Usage:

fitted_params = optimize(func, x_data, y_data, guess_params)
func - the model function to fit
x_data - array of x data points
y_data - array of y data points
guess_params - initial guess for model parameters

Returns: array of optimized parameter values

Example:

def exponential(x, a, b):
    return a * np.exp(b*x)

x_data = [1, 2, 3, 4, 5] 
y_data = [2, 4, 8, 16, 32]

params = optimize(exponential, x_data, y_data, guess_params=[1, 1]) 
# Returns optimized params for the exponential model

savitzky_golay

Applies Savitzky-Golay filter to smooth noisy data.

Usage:

savitzky_golay(y, window_size, order, x)
y - array of noisy y data
window_size - size of filter window
order - order of polynomial filter
x - array of x data points

Returns: smoothed data

Example:

# Your data and parameters
x = np.linspace(0, 10, 100)
y_noisy = np.sin(x) + np.random.normal(0, 0.2, 100)  # Noisy sine wave
window_size = 15
degree = 2

# Applying Savitzky-Golay filter and plotting
savitzky_golay(y_noisy, window_size, degree, x)

Usage:

missing_data_imputation(data)

Returns: imputed data

Example:

data = np.array([1, 2, np.nan, 4, np.nan, 6, 7, np.nan, 9])


missing_data_imputation(data)
# Tries different imputation methods on data

regression

Fits various regression models to data and evaluates them.

Usage:

regression(x, y)
x - independent variable data
y - dependent variable data

Returns: regression models and evaluation

Example:

# Example data (replace with your datasets)
x_data = np.array([1, 2, 3, 4, 5])
y_data = np.array([2, 4, 5, 4, 5])

# Perform the regression analysis and display statistics in order of R-squared scores
regression(x_data, y_data)

model_analysis

Computes regression model evaluation metrics.

Usage:

model_analysis(y_true, y_pred, x)
y_true - observed y data
y_pred - predicted y data
x - predictor variable data

Returns: evaluation metrics

Example:

# Sample data
np.random.seed(42)
observed_values = np.random.rand(100) * 10  # Random observed values
predicted_values = observed_values + np.random.randn(100)  # Adding random noise to create predicted values
predictor_variable = np.random.rand(100) * 20  # Random predictor variable values

# Assuming n = number of samples, k = number of parameters
n_samples = len(observed_values)
n_parameters = 2  # For demonstration, assuming a model with 2 parameters

# Using the model_analysis() function
evaluation_results = model_analysis(observed_values, predicted_values, predictor_variable, n=n_samples, k=n_parameters)
print("Evaluation Metrics:")
for metric, value in evaluation_results.items():
    print(f"{metric}: {value}")
