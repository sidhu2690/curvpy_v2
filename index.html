CurvPy Documentation

Installation

pip install curvpy

Usage

Regression Analysis


import numpy as np
from curvpy import datasleuth

# Example usage
x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 6, 8, 10])

# Perform regression analysis
results = datasleuth(x, y)
print(results)

Optimization


import numpy as np
from curvpy import optifit

# Define the function to fit
def func(x, a, b, c):
    return a * np.sin(b * x) + c

# Generate some example data
x_data = np.linspace(0, 10, 100)
y_data = func(x_data, 2.5, 1.3, 0.5) + np.random.normal(scale=0.5, size=len(x_data))

# Initial guess for parameters
guess_params = [1.0, 1.0, 1.0]

# Perform curve fitting and display results
fit_results = optifit(func, x_data, y_data, guess_params)
print(fit_results)

API Reference

curvpy.datasleuth(x, y)

Perform regression analysis on the given data points.


x (array-like): The independent variable data.
y (array-like): The dependent variable data.

Returns:
results (dict): A dictionary containing the results of the regression analysis.

curvpy.optifit(func, x, y, guess_params)

Perform optimization by fitting a function to the given data points.


func (callable): The function to fit.
x (array-like): The independent variable data.
y (array-like): The dependent variable data.
guess_params (list): The initial guess for the function parameters.

Returns:
fit_results (dict): A dictionary containing the results of the optimization.
