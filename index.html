CurvPy

Welcome to the CurvPy documentation! CurvPy is a powerful data analysis tool for performing regression analysis and optimization on datasets. This library is designed to simplify the process of analysing and optimizing data, making it accessible to beginners and experienced data scientists. CurvPy also has a UI-based repository, which has more utilities. (CurvPy is still under development)

Installation

pip install curvpy

Usage

Regression Analysis

CurvPy provides a simple interface for performing regression analysis on your data. Here's an example of how to use CurvPy for regression analysis:


import numpy as np
from curvpy import datasleuth

# Example usage
x = np.array([1, 2, 3, 4, 5])
y = np.array([2, 4, 6, 8, 10])

# Perform regression analysis
results = datasleuth(x, y)
print(results)

Optimization

In addition to regression analysis, CurvPy also offers an optimization module for fitting functions to your data. Here's an example of how to use CurvPy for optimization:


import numpy as np
from curvpy import optifit

# Define the function to fit
def func(x, a, b, c):
    return a * np.sin(b * x) + c

# Generate some example data
x_data = np.linspace(0, 10, 100)
y_data = func(x_data, 2.5, 1.3, 0.5) + np.random.normal(scale=0.5, size=len(x_data))

# Initial guess for parameters
guess_params = [1.0, 1.0, 1.0]

# Perform curve fitting and display results
fit_results = optifit(func, x_data, y_data, guess_params)
print(fit_results)

OptiFit_v2

In addition to OptiFit, we have another version of the same called OptiFit_v2, it also offers an optimization module for fitting functions to your data but it has some additional utilities. Here's an example of how to use CurvPy for optimization:


	    from curvpy import optifit_v2
import numpy as np

def func(x, a, b, c, d):
    return a * np.power(x, b) * np.exp(c * x + d)

x=np.array([10, 30, 50, 70, 90, 110, 130, 150, 170, 190, 210, 230, 250, 270])
y=np.array([4823, 6402, 6186, 5551, 5073, 4773, 4353, 4017, 3779, 3480, 3132])
initial_parameters = [0.0599344694,-0.0136083382,-0.0025016990,11.5658504714]

result = optifit_v2(func, initial_parameters, x, y)
print("Optimal parameters:", result)
